### 视频

- 视频可在线播放，部分视频可能内置播放器无法正常解码.
- 手机端(安卓)安装VLC后，点击视频下方的VLC橙色按钮,可以跳转到软件内正常播放.
- PC端安装PotPayer,点击视频下方的黄色PotPlayer按钮,可以跳转到PotPayer内正常播放.
